# 구현 기능 목록

## MVC 모델에 맞추어 구현


### Model (data)
- 다리 길이를 입력 받는다
- 위로갈지 아래로 갈지 입력 받는다.
- 다리 길이만큼 이동 할 수 있는 곳을 1과 0으로 랜덤하게 생성한다.
- 올바르지 못한 곳에 이동했을 경우 게임 재시작 여부를 입력받는다.
> 예외사항 
> - 다리 길이는 3~20 사이여야한다.
> - 다리 길이는 숫자여야한다.
> - 이동할 칸은 U, D 으로 입력되어야 한다.
> - 게임 재시작/ 종료 여부는 R, Q 으로 입력되어야 한다.

### View (UI)
- 사용자가 입력한 U,D에 맞춰 이동한걸 출력한다.
- 최종 게임 결과를 출력한다.
- 게임 시작한다.
- 게임 종료한다.

### Controller 
- 다리 길이를 전달
- 위, 아래 여부
- 게임 재시작 여부를 전달한다.
